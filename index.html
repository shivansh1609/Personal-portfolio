Crime Mapping and Analysis - Code


# Crime Mapping and Analysis using Python and Folium
# This script reads crime data from a CSV file, processes it, and generates an interactive map.

import pandas as pd
import folium
from folium.plugins import HeatMap, MarkerCluster

# 1. Load the dataset (replace 'crime_data.csv' with your file)
# The CSV should have 'Latitude' and 'Longitude' columns.
try:
    df = pd.read_csv('crime_data.csv')
    # 2. Data Cleaning: Drop rows with missing location data
    df.dropna(subset=['Latitude', 'Longitude'], inplace=True)
except FileNotFoundError:
    print("Error: crime_data.csv not found. Creating dummy data for demonstration.")
    data = {
        'Latitude': [28.6139, 28.5355, 28.6562, 28.7041],
        'Longitude': [77.2090, 77.2244, 77.2410, 77.1025],
        'CrimeType': ['Theft', 'Assault', 'Burglary', 'Robbery'],
        'Date': ['2023-01-15', '2023-02-20', '2023-03-10', '2023-04-05']
    }
    df = pd.DataFrame(data)

# 3. Initialize the map
# Centered on a sample location (e.g., Delhi, India)
map_center = [df['Latitude'].mean(), df['Longitude'].mean()]
crime_map = folium.Map(location=map_center, zoom_start=11, tiles="CartoDB dark_matter")

# 4. Create Feature Groups
heatmap_layer = folium.FeatureGroup(name='Crime Heatmap')
marker_cluster_layer = folium.FeatureGroup(name='Individual Incidents')

# 5. Add HeatMap layer
# Create a list of points for the heatmap
heat_data = [[row['Latitude'], row['Longitude']] for index, row in df.iterrows()]
HeatMap(heat_data, radius=15).add_to(heatmap_layer)

# 6. Add Marker Cluster layer for individual incidents
marker_cluster = MarkerCluster().add_to(marker_cluster_layer)

# Add markers for each crime incident
for idx, row in df.iterrows():
    popup_text = f"Crime: {row.get('CrimeType', 'N/A')}
Date: {row.get('Date', 'N/A')}"
    folium.Marker(
        location=[row['Latitude'], row['Longitude']],
        popup=popup_text,
        icon=folium.Icon(color='red', icon='info-sign')
    ).add_to(marker_cluster)

# 7. Add layers to the map
heatmap_layer.add_to(crime_map)
marker_cluster_layer.add_to(crime_map)

# 8. Add Layer Control to toggle between views
folium.LayerControl().add_to(crime_map)

# 9. Save the map to an HTML file
output_filename = 'crime_map.html'
crime_map.save(output_filename)

print(f"Interactive crime map has been generated: {output_filename}")

Posture Detection - Code

This project uses p5.js for drawing and webcam access, and ml5.js for the pre-trained PoseNet model. The following is a simplified example of the core logic in a `sketch.js` file.


// Posture Detection using p5.js and ml5.js
// This script requires p5.js and ml5.js to be included in your HTML file.

let video;
let poseNet;
let poses = [];
let status = 'Good';

function setup() {
  createCanvas(640, 480);
  video = createCapture(VIDEO);
  video.size(width, height);

  // Create a new poseNet method with a single detection
  poseNet = ml5.poseNet(video, modelReady);
  
  // This sets up an event that fills the global variable "poses"
  // with an array every time a new pose is detected
  poseNet.on('pose', function(results) {
    poses = results;
    checkPosture();
  });

  video.hide(); // Hide the video element, and just show the canvas
}

function modelReady() {
  console.log('Model Loaded!');
}

function draw() {
  image(video, 0, 0, width, height);
  
  // We can call both functions to draw all keypoints and the skeletons
  drawKeypoints();
  
  // Display posture status on the screen
  fill(status === 'Good' ? '#22c55e' : '#ef4444');
  textSize(32);
  textAlign(CENTER, CENTER);
  text(status + ' Posture', width / 2, 50);
}

// A function to draw ellipses over the detected keypoints
function drawKeypoints() {
  for (let i = 0; i < poses.length; i++) {
    let pose = poses[i].pose;
    for (let j = 0; j < pose.keypoints.length; j++) {
      let keypoint = pose.keypoints[j];
      if (keypoint.score > 0.2) {
        fill(56, 189, 248);
        noStroke();
        ellipse(keypoint.position.x, keypoint.position.y, 10, 10);
      }
    }
  }
}

// A simple function to check posture based on nose and shoulder position
function checkPosture() {
    if (poses.length > 0) {
        const nose = poses[0].pose.keypoints[0]; // Nose is the first keypoint
        const leftShoulder = poses[0].pose.keypoints[5];
        const rightShoulder = poses[0].pose.keypoints[6];

        if (nose.score > 0.5 && leftShoulder.score > 0.5 && rightShoulder.score > 0.5) {
            const shoulderY = (leftShoulder.position.y + rightShoulder.position.y) / 2;
            
            // If the nose is significantly below the average shoulder height, it's bad posture
            if (nose.position.y > shoulderY + 20) { 
                status = 'Bad';
            } else {
                status = 'Good';
            }
        }
    }
}

Sales Analytics Dashboard - Details

This project was developed using Microsoft Excel and Power BI, which are visual analytics tools rather than traditional coding environments. The process involved data cleaning, transformation, and visualization.

Process in Excel:

  • Data Cleaning: Removed duplicates, corrected data types, and handled missing values using Power Query.
  • Feature Engineering: Created new columns like 'Month', 'Year', and 'Profit Margin' using Excel formulas. Example: =B2-C2 for profit.
  • Pivot Tables: Created pivot tables to summarize data, such as total sales by region or product category.

Process in Power BI:

  • Data Modeling: Imported cleaned data from Excel and established relationships between different tables (e.g., Sales and Products).
  • DAX Measures: Wrote DAX (Data Analysis Expressions) formulas to create dynamic calculations. Example: Total Sales = SUM(Sales[Revenue]).
  • Visualization: Created interactive charts, graphs, and maps. Added slicers and filters to allow users to drill down into the data.
  • Dashboard Design: Assembled the visualizations into a cohesive and easy-to-understand dashboard layout.
